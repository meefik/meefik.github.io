This application is open-source software for downloads and updates time zones to the latest version on your Android device. This update should fix all known problems with time zones, such as incorrect time in Android and some applications.

Requirements:

  • Android 4.0 (API 14) or later
  • Superuser permissions (root)
  • BusyBox

Update procedure:

  1. Get superuser privileges (root)
  2. Install BusyBox
  3. Check your connection to the Internet
  4. Tap UPDATE button
  5. Restart your device

The following files will be updated:

  • TZ data: /data/misc/zoneinfo/tzdata or /system/usr/share/zoneinfo/*
  • ICU data: /system/usr/icu/*.dat

Attention! Before starting the update procedure is recommended to make backup copies of these files or the entire system.
